Это программа является графическим инструментом для взаимодействия с программой yt-dlp

Ссылки

https://github.com/yt-dlp/yt-dlp
https://ffmpeg.org/

В архив релиза включены исполняемые файлы yt-dlp и ffmpeg (они НЕ являются частью этой программы), вы можете использовать другие файлы.

Как пользоваться:
распаковать архив и запустить FFGui.exe
